# eRealm - 乙域旅游指南项目说明文档

## 项目简介

eRealm是一个呈现阈限空间和亚文化美学的3D交互式网页，融合了低分辨率高对比度复古风格与现代3D技术，为用户提供沉浸式的场景探索体验。

## 作品介绍

### 技术栈
- **HTML5**：基础结构搭建
- **CSS3**：样式设计，包括网格背景、光效和动画
- **JavaScript**：交互逻辑，实现鼠标光效、菜单控制等功能
- **Three.js**：3D场景渲染，构建沉浸式空间

### 功能特性

#### 视觉效果
- 低分辨率/高对比度复古风格设计
- 黑白主色调搭配红色强调元素，营造独特氛围
- 黑白相间的网格背景，随鼠标悬停产生变形效果
- 高饱和度鼠标光效，带有渐隐动画

#### 交互功能
- 鼠标光效跟随，边缘粗糙的高饱和度视觉效果
- 鼠标移动时光效渐隐：色彩减淡并缩小
- 场景选择按钮，带有红色流动光效
- 场景内折叠菜单，包含返程、光效控制和使用说明

#### 3D场景
- **走廊**：随机场景，带用户进入未知空间
- **海**：池核场景，营造宁静而神秘的氛围
- **教室**：梦核场景，唤起怀旧与超现实感

### 使用方法
1. 在浏览器中打开`index.html`文件
2. 点击场景选择按钮进入不同的3D场景
3. 在场景中使用左上角菜单进行设置
4. 使用鼠标/触摸控制视角移动，探索场景

## 目标人群

- 对阈限空间、梦核、池核等亚文化美学感兴趣的用户
- 喜欢复古风格、低像素艺术的互联网用户
- 追求沉浸式3D交互体验的探索者
- 对数字艺术、新媒体艺术有研究或兴趣的人群

## 解决什么问题

1. **亚文化美学展示**：为阈限空间、梦核、池核等小众亚文化提供了一个可视化的展示平台，让更多人了解和体验这种独特的美学风格。

2. **沉浸式交互体验**：通过3D技术和精心设计的交互效果，创造出一个能够让用户完全沉浸其中的虚拟空间，满足用户对探索和体验的需求。

3. **复古风格与现代技术融合**：将低分辨率、高对比度的复古设计风格与现代3D渲染技术相结合，创造出一种独特的视觉体验，填补了传统网页设计与纯3D游戏之间的空白。

4. **简单易用的操作界面**：提供了直观的场景选择和菜单控制，让用户能够轻松上手，专注于场景探索和体验，无需复杂的操作学习。

5. **情感共鸣与心理体验**：通过精心设计的场景氛围和交互效果，能够唤起用户的情感共鸣，提供一种独特的心理体验，满足用户对情感表达和心理探索的需求。
